# Portuguese Compiler

## What is this?
****

This is a compiler for a Portuguese-Like programming language for language processing curricular unit.

## How to test it?
****

You can test this compiler using ocamlbuild the following way:

```
	ocamlbuild -use-menhir main.native
```
